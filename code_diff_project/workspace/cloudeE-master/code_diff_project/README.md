# Code Diff Project

This project contains a Django backend and a React frontend.

## Structure

- `backend/`: Django project
- `frontend/`: React application
- `venv/`: Python virtual environment

## Setup

1.  **Backend**:
    ```bash
    ./venv/Scripts/activate
    cd backend
    python manage.py runserver
    ```

2.  **Frontend**:
    ```bash
    cd frontend
    npm start
    ```
